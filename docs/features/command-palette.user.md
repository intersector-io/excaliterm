# Command Palette

A keyboard-driven launcher for common actions and fast navigation.

## Opening

Press **Cmd+K** (macOS) or **Ctrl+K** (Windows/Linux) anywhere in the app.

## Using it

- Type to filter commands.
- Use **↑ / ↓** to move the selection, **Enter** to execute, **Esc** to close.
- Commands are grouped into **Create**, **Navigate**, **Actions**, and **Terminals**.

## Available commands

- **Create** — New Terminal, New Note, (and more as the app grows).
- **Navigate** — Go to Canvas, Go to Chat, etc.
- **Actions** — workspace-wide actions like Auto Layout, Close All Terminals.
- **Terminals** — a dynamic entry per terminal so you can jump to any one by ID or tag.

Commands that aren't currently available (for example, creating a terminal when no host is online) are shown disabled with a hint.
