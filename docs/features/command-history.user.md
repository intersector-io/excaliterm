# Command History

Every command you run in a terminal is captured and browsable from a dedicated history panel.

## Opening command history

From a terminal's overflow menu choose **Command History**. A linked history node appears on the canvas next to the terminal. On mobile, it's accessible from the terminal's back face (flippable card).

## What's shown

Two tabs:

- **History** — every command in chronological order with execution time.
- **Top 10** — the ten most frequently used commands with their count and last-executed timestamp.

## Per-command actions

Each command row has two buttons:

- **Copy** — copies the command text to your clipboard.
- **Execute** — re-sends the command to the originating terminal.

## Scope

History is scoped per-terminal. Dismissing or closing a terminal removes its associated history.
