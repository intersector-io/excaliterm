# Sticky Notes

Notes are lightweight markdown sticky notes that live on the canvas (or in a dedicated mobile section). Use them for context, checklists, credentials, links, or anything you want to share with other collaborators.

## Creating a note

- **Desktop canvas**: click **New Note** in the canvas controls, or invoke it from the command palette. A note node appears at 300×300 by default.
- **Mobile**: open the Notes section and tap **New Note**.

## Editing

- **Desktop**: click the note to edit inline; markdown renders in preview mode.
- **Mobile**: tap a note to open a fullscreen markdown editor. Back button returns to the list.

Changes are saved automatically with debounced API calls.

## Managing notes

- **Desktop**: drag the node to move it, drag corners to resize, use the × button in the node header to delete.
- **Mobile**: tap the trash icon on the note card to delete; previews show the first three lines.
