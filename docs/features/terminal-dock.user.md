# Terminal Dock (Desktop)

The **Terminal Dock** runs along the bottom of the canvas on desktop. It shows miniature "skeleton" cards for every terminal in the workspace, letting you find and jump to any terminal without panning the canvas.

## Grouping

Toggle the grouping mode:

- **Tag** — terminals grouped by their first tag
- **Host** — terminals grouped by the host they run on
- **Status** — active / disconnected / exited / error

## Searching

The search input filters skeleton cards in real time by terminal ID or tag.

## Interacting

- **Click a skeleton** — pans the canvas so that terminal is centered and focused.
- **Double-click** — opens the terminal fullscreen.
- **Collapse/expand** — chevron button on the dock header hides or reveals the cards.

## Mobile

The dock is desktop-only. The mobile equivalent is the [mobile terminal list](./mobile-experience.user.md), which has its own grouping and filtering controls.
