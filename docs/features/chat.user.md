# Chat

Every workspace has a real-time chat channel for all collaborators.

## Opening chat

- **Desktop**: click the Chat item in the sidebar, or toggle the slide-out chat panel with **Cmd+Shift+C** (Ctrl+Shift+C).
- **Mobile**: tap the **Chat** tab in the bottom navigation for a full-screen view.

## Using chat

- Type a message and press **Enter** to send.
- Messages show sender display name, content, and timestamp.
- The view auto-scrolls to the latest message on new arrivals.
- Scroll up to load older messages; they load 50 at a time.
- An unread badge appears on the Chat tab / icon when new messages arrive while the panel is closed.

## Display name

Your chat messages use your **collaborator display name**. You can change it any time from the sidebar or workspace header — see [collaboration.user.md](./collaboration.user.md).

## Persistence

Messages are stored server-side and visible to anyone who opens the workspace URL.
