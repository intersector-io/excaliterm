# Split Terminal View (Desktop)

In desktop fullscreen mode, you can split the view to work with multiple terminals side by side.

## Layouts

The split controls in the fullscreen header offer:

- **H Split** — two terminals side by side
- **V Split** — two terminals stacked vertically
- **Quad** — 2×2 grid of four terminals

## Selecting terminals per pane

Each pane has its own dropdown listing all terminals in the workspace. Pick any terminal for any pane; you can even show the same terminal in multiple panes.

## Exiting split

Click the single-pane button in the header to collapse back to a single terminal.

## Availability

Split view is desktop-only. On mobile, swipe left/right in fullscreen to cycle between terminals instead.
